## :bookmark: Sobre

Move-it é uma plataforma para ajudar pessoas as pessoas a se se alongarem. Este projeto foi implementado durante a **Next Level Week #4** da **[Rocketseat](https://rocketseat.com.br/)**.

<br>

## :computer: Tecnologias

-  **[Typescript](https://www.typescriptlang.org/)**
-  **[ReactJS](https://reactjs.org/)**

<br>

## :wrench: Ferramentas

- **[VisualStudio Code](https://code.visualstudio.com/)**
- **[Google Chrome](https://www.google.com/chrome/)**

<br>

## :package: Instalação

### :heavy_check_mark: **Pré-requisitos**

Os seguintes softwares devem estar instalados:
  
  - **[Git](https://git-scm.com/)**
  - **[NPM](https://www.npmjs.com/)** or **[Yarn](https://yarnpkg.com/)**

<br>
  
### :arrow_down: **Clonando o repositório**

```sh
  $ git clone https://github.com/GabrielSouzaCampos/MoveIt.git
```
